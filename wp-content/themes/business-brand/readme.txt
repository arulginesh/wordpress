== Theme Description ==
Business Brand is a premium quality WordPress theme which can be used for literally any type of website. Business Brand theme is a mobile responsive WordPress theme. It supports SiteOrigin page builder and hence any type of page layout can be easily designed. Business Brand is a language ready theme hence it can be translated to any type of LTR language. Logo, colors etc can be easily managed in the WordPress customizer settings area. Step by step documentation and dummy data is also available for free at: https://sigmathemes.com/documentation/business-brand/

== Installation ==
1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "business-brand.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (business-brand.zip) that you have downloaded.
 = Upload the entire folder (business-brand) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/business-brand/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "Business Brand" to activate the theme.

5. Complete all of the required inputs in the Appearance => Custimize option and click "Save & Publish" to make the changes LIVE.
